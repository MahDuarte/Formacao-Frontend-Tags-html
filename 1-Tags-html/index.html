Títulos e Subtítulos

topo

Tags h1 a h6: podemos utilizar para definir diferentes tamanhos de letra para títulos e subtítulos.

Adicionando Parágrafos

topo

Tag p: usamos a tag "p" quando queremos adicionar parágrafos ao nosso texto.

Gerando um link

topo

Tag a: às vezes precisamos encaminhar o usuário para outra página através de um link, esse carinha pode fazer isso pra gente.

Excluindo partes do texto

topo

Tag del: ops! tem algo que quer, mas não quer deixar no seu site? pode usar a tag del, ela irá adicionar um efeito de tachado nas palavras que quiser.

Abreviaturas e Siglas

topo

Tag abbr: essa tag cria siglas ou abreviações, o usuário irá ver a sigla, mas quando parar com o mouse sobre a sigla, uma caixinha com o significado dela irá APRCR.

Criando Listas

topo

Falando em listas, temos dois tipos;

Tag ol: a tag ol irá criar listas ordenadas, ou seja, em ordem.

Tag ul: o outro tipo é a lista não ordenada, ela não segue uma ordem específica.

Tag li: cada lista é composta por itens, esses itens são criados a partir da tag li.

Aplicando Estilos

topo

Tag strong: essa tag é bem simples, ela irá aplicar negrito ao nosso texto.

Tag i: se quiser usar o itálico, essa é a tag para isso.

Tag u: precisa dar ênfase em algo importante? Pode deixar sublinhado com a tag u.

Tag mark: ok! tem algo ainda mais importante no seu texto, que tal usar o famoso marca texto?

Tag sub: precisando escrever a fórmula de algum elemento QUÍMICO? sem problemas, pode usar a tag sub, ela vai deixar o elemento um pouco abaixo da linha de texto.

Tag sup: expressões matemáticas às vezes pedem uma raiz2, a tag sup pode ajudar, ela deixará o elemento um pouco acima da linha de texto.

Tag font*: essa tag define a fonte do seu texto, através dela você pode alterar três aspectos: o tipo de fonte, a cor e o tamanho dela.

Tag small: a tag small define um texto para um tamanho menor dentro do seu texto, podemos considerar aquelas entrelinhas.

Tag hr: quer deixar uma linha horizontal simples para separar conteúdos na sua página? essa tag funciona muito bem.

* a tag font não é mais suportada no HTML5.
